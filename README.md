# Hospital-Management-System🏥

A simple Java-based Hospital Management System developed using Object-Oriented Programming (OOP) concepts. This project helps manage patients, doctors, and appointments through a menu-driven console application.

🚀 Features
Add new patients
Add doctors with specialization
Book appointments between patients and doctors
View patient records
View doctor records
Simple and user-friendly console interface
🛠 Technologies Used
Java
OOP Concepts
Abstraction
Inheritance
Polymorphism
Encapsulation
ArrayList Collection
Scanner Class for User Input
📚 Project Concepts

This project demonstrates:

Abstract Classes
Method Overriding
Dynamic Data Storage using ArrayList
Menu-driven Programming
Basic Hospital Record Management
📂 Project Structure
Person → Abstract base class
Patient → Stores patient details
Doctor → Stores doctor details and specialization
Appointment → Manages appointments
HospitalManagementSystem → Main execution class
▶️ How to Run
Install Java JDK
Open project in any IDE (Eclipse / IntelliJ / VS Code)
Compile the program
Run HospitalManagementSystem.java
📸 Output

The program provides a console-based interface for managing hospital operations efficiently.
